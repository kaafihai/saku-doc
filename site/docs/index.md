# Welcome to Saku Docs

Saku Docs turns your OpenAPI spec and Markdown files into a fast, beautiful static documentation site.

## Features

- **OpenAPI reference** — full endpoint, parameter, and schema rendering
- **Markdown pages** — write guides, tutorials, and reference docs
- **Sidebar navigation** — automatically generated from your file structure
- **Static output** — deploy anywhere, no server required

## Quick Start

1. Add your OpenAPI spec to `saku.config.json`
2. Create Markdown files in your `docsDir`
3. Run `pnpm dev` to preview

Browse the **Guide** section in the sidebar to get started, or jump straight to the [API Reference](/api).
