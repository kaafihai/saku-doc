---

Point it at an OpenAPI spec, a folder of Markdown files, or both, and Saku Docs builds a polished, static documentation site ready to deploy anywhere.

## Get Started

```bash
npm install -g saku-doc
saku-doc build -d ./my-docs
```

That's it. Check out the [Getting Started guide](/docs/guide/getting-started) for full details.
