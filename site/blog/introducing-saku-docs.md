pubDate: 2026-03-14
description: Saku Doc is a fast, zero-config documentation generator built on Astro. Turn your OpenAPI spec and Markdown into a beautiful static site in seconds.
---

Point it at an OpenAPI spec, a folder of Markdown files, or both, and Saku Doc builds a polished, static documentation site ready to deploy anywhere.

## Get Started

```bash
npm install -g saku-doc
saku-doc build -d ./my-docs
```

That's it. Check out the [Getting Started guide](/documentation/) for full details.
